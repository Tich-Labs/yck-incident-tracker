3. Incident Workflow

New → Assigned → PFA In Progress → Under Review → Escalated → Resolved → Closed

✅ Assignment

Staff assign cases to counselors based on type and location

📝 Notes

Timestamped notes track all case activity

🔼 Escalation

Complex cases escalate to Program Leads

📧 Notifications

Email alerts at every workflow stage

⚡ Workflow Tip

Use the status tabs on the Incidents page to filter cases by stage.
