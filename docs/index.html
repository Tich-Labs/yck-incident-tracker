4-step wizard with large tap-friendly icon cards for incident types.

📝 10 Incident Types

Physical, Sexual, Emotional, Neglect, Bullying, Domestic Violence, Child Exploitation, etc.

4. Submission Success

Confirmation with checkmark and confidentiality assurance.

✅ Anonymous

Survivors get a reference code (e.g., NT111974) but no PII is collected.

1. Dashboard

Personalized greeting with role badge, summary cards, incidents awaiting assignment.

Filterable list with status tabs (New, Assigned, In Progress, Escalated, etc.).

🔍 Filter & Search

Category icons, location, date, status badges. Click row for details.

3. Reports & Analytics

Anonymized analytics with time-range filters, CSV export.

📊 Data Anonymization

All exports automatically anonymize survivor data. No PII in reports.

4. User Management

Search, filter, approve pending users, change roles, deactivate accounts.

👥 Staff Workflow

New staff sign up → Status "Pending" → Admin approves → Role assigned

🏥 Kakamega & Vihiga

County filters, service categories, active/inactive status management.

6. Admin Manual

In-app documentation with sidebar navigation for all operational workflows.

📖 Self-Service

Overview, RBAC, Incident Workflow, Reports, Privacy, Audit Log, Support.
